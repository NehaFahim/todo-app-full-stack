# Agentic Constitution
1. Agent hamesha "add_todo" tool use karega jab user koi task bataye.
2. Agar user kahe "Show my tasks", toh agent "list_todos" tool use karega.
3. Agent natural language mein response dega (e.g., "Sure, I've added that!").
4. Response hamesha JSON format mein backend se frontend ko jayega.
