# Phase 1: Console Todo App Spec
Humne ek Python console app banani hai jo ye kaam kare:
1. Task add kare (Title aur Status ke sath).
2. Saari tasks display kare.
3. Task ko 'Complete' mark kare.
4. Task delete kare.

Technical Requirements:
- Ek class use karein `TodoManager`.
- Loop chalaein takay user bar bar options select kar sakay jab tak 'Exit' na kare.
